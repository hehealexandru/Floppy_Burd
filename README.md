Welcome to the GitHub repository for Floppy Burd, a creative and addictive version of the classic game Flappy Bird. Passionately developed in Unity and meticulously programmed in C#, this game promises to provide hours of fun and challenges.

# 🐦 Floppy Burd

**Floppy Burd** is an arcade-style game that challenges players' agility and patience. Control an adorable bird as you navigate through colorful, dynamic obstacles in a vibrant and lively world!

---

## 🎮 Game Mechanics

Simple yet addictive:

- **Tap the screen or use keyboard keys** to make your bird flutter.
- Skillfully navigate through incoming obstacles.
- Aim for the highest score possible by avoiding collisions!

---

## 🛠️ Project Code Structure

The game utilizes several **C# scripts** working seamlessly to deliver a smooth gameplay experience:

- 📜 **`burd_script.cs`**: Handles bird movement, allowing it to `fly` and interact dynamically with obstacles.
- 📜 **`pipe_spawner.cs`**: Dynamically generates and randomizes obstacle positions.
- 📜 **`pipe_script.cs`**: Controls obstacle speed and removes obstacles once they're out of the gameplay area.
- 📜 **`Middle_Score_script.cs`**: Manages the scoring system, incrementing scores each time an obstacle is successfully avoided.
- 📜 **`logic_manager.cs`**: Core script managing game states (start, running, game over) and UI interactions.

## 🖼️ Assets & Graphics

- Carefully selected graphic assets provide a vibrant and immersive gaming environment.

---

## 🎯 Scoring and Game Over

- Score points by successfully navigating past obstacles.
- Colliding with an obstacle or hitting the ground triggers the game over state.
- Upon game over, players have the immediate option to retry and improve their score.
